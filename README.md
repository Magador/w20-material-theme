# W20 Material Theme

## Installation

```
bower install w20-material-theme
```

## Usage

### Configuration

```json
"bower_components/w20-material-theme/w20-material-theme.w20.json": {
  "modules": {
    "sidenav": {
      "logoUrl": "/home",
      "logoImg": "/images/logo.png",
      "backgroundImg": "/images/header-background.jpg"
    },
    "main": {
      "theme": {
        "primaryPalette": {
          "name": "indigo"
        },
        "accentPalette": {
          "name": "amber"
        }
      }
    }
  }
}
```
